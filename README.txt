Это решение разного рода задач по парпрогу для студентов 313 группы МФТИ
Автор - Латушко С.А.

Для не знакомых с гитом: клонировать репозиторий можно следующей командой, после использования которой в вашей текущей директории появится поддиректория par_prog с содержимым:

[noob]$ git clone https://github.com/StasLatGTTT/par_prog.git


Компилировать можно из командной строки Linux с использованием скрипта compile.sh
Аргумент скрипта - имя файла в директории src без расширения
Пример использования:

[noob]$ ./compile.sh hello_world
[noob]$ ./hello_world
Hello world!
Hello world!
Hello world!
Hello world!
[noob]$

 - сказать спасибо (я буду рад);
 - принести шоколадку/пачку чипсов с колой (будет здорово, если не слишком часто, ибо объемся);
 - поделиться в ответ чем-то ещё (теорфизом, например - буду очень рад);
 - если вас препод прижмёт к стенке и спросит "ЧЕЙ КОД УКРАЛ МРАЗЬ?!", вы честно признаетесь, что взяли у офигенного меня (моё мнение о вас будет как об образце чести и справедливости)ж
 - указать на ошибку или недоработку (защищу перед преподом как умного человека, но только если указали вежливо).
