Это решение разного рода задач по парпрогу для студентов 313 группы МФТИ
Автор - Латушко С.А.

Для не знакомых с гитом: чтобы получить у себя папку par_prog со своим
содержимым в текущей директории, наберите следующую команду:

[noob]$ git clone https://github.com/StasLatGTTT/par_prog.git

Самый простой способ скомпилировать все существующие задания - в терминале Linux
выполнить make, находясь в каталоге проекта. Все исполняемые файлы окажутся в
каталоге bin. Для удаления исполняемых файлов (и временных, если таковые будут)
исполнить make clean.

Также компилировать можно из командной строки Linux с использованием скрипта
compile.sh (устаревший метод). Аргумент скрипта - имя файла в директории src
без расширения. Исполняемые файлы также окажутся в директории bin.
Пример использования:

[noob]$ ./compile.sh hello_world
[noob]$ ./bin/hello_world
Hello world!
Hello world!
Hello world!
Hello world!
[noob]$
