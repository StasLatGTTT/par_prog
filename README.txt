Это решение разного рода задач по парпрогу для студентов 313 группы МФТИ
Автор - Латушко С.А.

Компилировать можно из командной строки Linux с использованием скрипта compile.sh
Аргумент скрипта - имя файла в директории src без расширения
Пример использования:
[noob]$ ./compile.sh hello_world
[noob]$ ./hello_world
Hello world!
Hello world!
Hello world!
Hello world!
[noob]$

Я делаю это по доброте душевной, но если вам понравится результат, то отблагодарить меня можно следующими путями:
 - сказать спасибо (я буду рад);
 - принести шоколадку/пачку чипсов с колой (будет здорово, если не слишком часто, ибо обожрусь)ж
 - поделиться в ответ чем-то ещё (теорфизом, например - буду очень рад);
 - если вас препод прижмёт к стенке и спросит "ЧЕЙ КОД УКРАЛ МРАЗЬ?!", вы честно признаетесь, что взяли у офигенного меня (моё мнение о вас будет как об образце чести и справедливости).
